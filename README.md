# 社員＆商品管理システム

## 概要
このシステムでは、社員情報、出退勤情報、商品情報を一元管理する事ができます。
社員の新規登録から編集・削除。
出退勤の入力。
商品の新規登録から編集・削除、社員からの商品レビューも入力できます。

## 主な機能
- ログイン・ログアウト機能
- 新規社員登録画面
- 出退勤登録画面
- 出退勤一覧画面、時刻編集
- 社員一覧、編集、削除(管理者のみ)
- 商品一覧、新規登録、編集、削除機能
- 商品レビュー画面

## 開発背景<br>
どのようなシステムを開発するか考えた時に業務管理システムの多重構造があるのではないかという課題があがりました。<br>
- 社員、商品、出退勤などの情報が別の情報管理システムを用いて管理されているため、データが連動していない事。<br>
- どれかの情報に更新があった場合、他の情報も手動で更新する必要があり、時間が取られている。<br>
- 取り扱い商品について、社員の率直な意見を吸い上げる場がない。<br>
### そこで社員、商品、出退勤の情報を一元化<br>
- 一元化により各種コストの削減
- 一元化により様々な情報がリアルタイムで反映され、社内での正確で素早い情報共有が実現。<br>
- 商品に関する、社員の率直な意見を吸い上げる場が生まれることで、商品に対するクレームなどの社内共有や柔軟な戦略を立てることが可能になるのではないかという事で商品レビュー機能をつけました。
