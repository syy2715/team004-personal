# 社員＆商品管理システム

## 概要
このシステムでは、社員情報、出退勤情報、商品情報を一元管理する事ができます。
社員の新規登録から編集・削除。
出退勤の入力。
商品の新登録から編集・削除、社員からの商品レビューも入力できます。

## 主な機能
- ログイン・ログアウト機能
- 新規社員登録画面
- 出退勤登録画面
- 出退勤一覧画面、時刻編集
- 社員一覧、編集、削除(管理者のみ)
- 商品一覧、新規登録、編集、削除機能
- 商品レビュー画面

## 開発環境
PHP 8.2.12<br>
MySQL互換（MariaDB 10.4.32）<br>
Laravel 10.50.0
